Linux系统学习总结

1.linux常用命令：

  • linux命令模式：命令 参数1 参数2 参数3 …. 命令作用对象

  • who – 查看谁登录了系统 / who -q – 统计登录人数

  • clear – 清楚屏幕输出

  • cal – 查看万年历 （calendar）

  • date – 查看时间日期

  • whatis 命令 – 查看命令描述

  • whereis 命令 / which 命令 – 获取命令的路径

  • shutdown – 关机或重启服务器

  • logout / exit– 退出登录服务器

  • ps – 查看进程信息 / ps aux – 查看所有进程信息

  • history – 查看历史命令 / history -c – 清除历史命令

  • !! – 再次执行上个命令

  • !历史命令编号 – 再次执行该编号对应的命令

  • write 用户名– 给其他登录系统的用户发消息

  • wall – 给所有人发消息

2.Linux常用快捷键

  • tab – 命令补全或路径补全

  • Ctrl + c – 终止一个命令的执行 / Ctrl + z – 把一个正在执行的命令暂停并放到后台，用 jobs 命令可查看，用

  • kill %命令编号 可终止后台命令

  • Ctrl + w – 删除一个单词 / Ctrl + u – 删除整行

  • Ctrl + a – 光标回到行首 / Ctrl + e – 光标回到行尾
  • Ctrl + d – 结束输入

3.Linux文件管理

  • pwd – 显示当前的工作目录 （print working directory）
    • root用户的根目录 – /root
    • 普通用户的根目录 – /home/用户名
  • cd 路径 – 切换到该路径（和windows的cmd里的命令一样，./ – 当前目录，.. – 当前路径的上级目录）
  • Linux根目录下的几个重要文件：
    • /root – 超级管理员的用户主目录
    • /home – 普通用户的用户主目录
    • /etc – 配置文件
    • /usr – 可执行的命令以及用户安装的程序
    • /var – 程序运行中产生的数据和日志
    • /proc – 进程文件
      • （CPUinfo and Memoryinfo）
  • ls – 显示当前目录文件列表 （list directory contents）
    • ll / ls -l – 长格式查看当前目录文件列表 （lone）
    • ls -a / ll -a – 显示所有文件，包括隐藏文件（all）
  • cat – 连接多个文件或查看文件内容 / cat -n – 查看文件内容是显示行号 / od – 以二进制查看文件
  • mkdir – 创建文件夹（make directory）
  • rmdir – 删除空文件夹（remove directory）
  • touch – 创建空文件或改变文件的最后访问时间
  • rm – 删除文件或文件夹
    • rm -i – 交互式删除（interactive）
    • rm -f – 强制删除（force）
    • rm -r – 递归式删除，与-f参数结合使用可删除文件夹（recursive）
  • cp 待拷贝文件路径 目标路径 – 拷贝文件
    • cp -r – 递归式拷贝文件，用于拷贝文件夹（recursive）
  • mv 待移动文件路径 目标路径 – 移动文件
  • file – 查看文件性质
  • wc – 统计文件字符数、单词数、行数
    • wc -l – 统计行数（line）
    • wc -w – 统计单词数（word）
    • -c – 统计字符数（character）
  • uniq – 去重 （unique）[只是在终端显示结果中去重，且只有相邻的重复会被抹掉，实际文件内容没变，可以用输出重定向来实现对文件去重]
    • uniq -c –统计重复次数（count）
  • sort – 排序
    • sort -r – 降序（reverse）
    • sort -n – 按数值排序（number）
  • iconv -f 原始编码 -t 目标编码 文件路径 – 编码转换（form、to）
  • chmod – 修改文件权限
    • chmod +x 文件名 – 给所有用户加上对该文件执行权限
    • chmod g+x 文件名 – 只给同组用户加上对该文件执行权限
    • chmod o+x 文件名 – 只给其他用户加上对该文件执行权限
    • chmod u+x 文件名 – 只给本用户加上对该文件执行权限
    • 其他两个权限修改同上，其他两个权限是r（读权限）和w（写权限）

4.Linux搜索命令

  • grep 字符串 – 搜索内容
  • find – 查找文件
    • find -name 文件名– 根据文件名查找
    • find -size 文件大小 – 根据文件大小查找，单位是KB
    • find -atime 时间 / find -ctime 时间/ find -mtime 时间– 根据时间查找

5.Linux网络相关命令

  • wget 网络地址– 网络下载器
  • scp – 跨网络安全文件拷贝（security copy）
  • sftp – 安全文件传输

6.Linux文本编辑器——vim

  • vim有三种模式：命令模式、编辑模式、底线命令模式

  • vim常用命令
    • 0 / $ / w – 行首 / 行尾 / 下一个单词
    • G / gg – 文件最后 / 文件开头
    • Ctrl + e / Ctrl + y – 翻行
    • Ctrl + f / Ctrl + b – 翻页
    • dd / d0 / d$ /dw – 删除一行 / 删除到行首 / 删除到行尾 / 删除一个单词
    • u / Ctrl +r – 撤销 / 重做
    • yy / p – 复制 / 粘贴
    • ZZ / ZQ / :wq / q / w / q! / qa– 保存退出 / 不保存退出 / 保存退出 / 不保存退出 / 保存 / 强制不保存退出 / 退出所有用vim打开的文本
    • :/字符 – 在文本中搜索该字符
    • 1,$s/字符1/字符2 – 将文本中所有的字符1替换为字符2
    • :set nu – 设置行号
    • :set hls – 设置高亮
    • :set relur – 设置标尺
    • :set ts=4 – 设置制表键（Tab）的空格数
    • set expandtab – 把制表键变成空格
    • set autoindent – 设置自动缩进
    • syntax off / syntax on – 设置语法高亮关 / 开

​ 进程池的概念和线程池一样，进程池就是多个子进程的管理工具，不同之处在于线程池里的每个线程可以访问并操作同一个数据，而进程池不行，进程池在创建时，会在每个子进程里拷贝一份主进程的数据，子进程只能操作拷贝的数据，无法操作主进程的数据
